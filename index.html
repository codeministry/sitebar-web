Εισάγετε τους σελιδοδείκτες από τον browser σας

Το να εισάγετε τους σελιδοδείκτες σας στο sitebar είναι πολύ εύκολο. Το sitebar μπορεί να εισάγει σελιδοδείκτες από ποικιλία μορφών σελιδοδεικτών.
Γενικά χρειάζεται πρώτα να κάνετε εξαγωγή τους σελιδοδείκτες σας και να τους αποθηκεύσετε σε ένα αρχείο.

Ενσωματώστε το sitebar στον browser σας

Υπάρχουν πολλοί τρόποι για να ενσωματώσετε το sitebar στον browser σας. To sitebar.gr προτείνει ανεπιφύλακτα την χρήση του Mozilla Firefox και της επέκτασης SiteBar Client που έχει σχεδιαστεί για να λειτουργεί για την υπηρεσία sitebar.
